How to create a webiste for free

Written By

Sergio Nicolini

Hi everyone today we are going to see how to create a website for free.

Wix

Wix is perhaps the most well-known site builder in the world and also the simplest in this list.

Pros

• Simple and good design sites

Cons

• Has a lot of limitations in the free version (under wix domain and wix advertising banner)

Total

• 6/10

Weebly

Weebly is a lesser known site builder than wix with simple and functional interface but has less limitations in the free version.

Pros

• Simple and less limitations than wix

Cons

• Few limitations (weebly sub domain and weebly advertising banner less invasive than that of wix)

Total

6/10

Nicepage + Netlify + Freenom

This alternative is more complicated than the others, in fact you must first create a website with nicepage, then export it to HTML and upload it to Netlify through this link, then register a free domain on Freenom and connect it to Netlify.

Pros

• No intrusive watermarks and full control over the sit

Cons

• Difficult to set up and need to renew the domain once a year

Total

• 7/10

WordPress + ProFreeHost + Freenom

Through these three services you can create a website in a more difficult but more professional way. To start you need to create an account on ProFreeHost and install WordPress, then install a page builder eg Brizy and if you want to create a blog I recommend installing a WordPress theme such as those of alx media and connect the domain previously registered on Freenom.

Pros

• No watermark, professional website and ability to create a blog without much difficulty

Cond

• Difficult to set up and need to renew the domain once a year

Total

• 7.5/10

Extra: Zyro

Zyro is a page builder where you can create a professional website quickly and easily. Unfortunately, however, this alternative is paid but the prices are low, in fact we talk about 9.49 euros per month for the basic version but it is often discounted to a few euros per month (currently 2.90 euros per month).

Pros

• Simple, fast and cheap

Cons

• Pricy

Total

• 9.5/10
